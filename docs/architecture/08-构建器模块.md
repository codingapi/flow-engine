# 08 - 构建器模块

## 模块概述

**包路径**: `com.codingapi.flow.builder`

构建器模块采用建造者模式，用于构建复杂的流程对象、节点对象和表单元数据。

## 构建器列表

| 构建器 | 说明 |
|--------|------|
| `WorkflowBuilder` | 工作流构建器 |
| `BaseNodeBuilder` | 节点构建器基类 |
| `ActionBuilder` | 动作构建器 |
| `FormFieldPermissionsBuilder` | 表单字段权限构建器 |
| `NodeMapBuilder` | 节点 Map 构建器 |
| `NodeStrategyBuilder` | 节点策略构建器 |
| `WorkflowStrategyBuilder` | 工作流策略构建器 |

## WorkflowBuilder

```java
package com.codingapi.flow.builder;

/**
 * 工作流构建器 - 建造者模式
 */
public class WorkflowBuilder {

    private final Workflow workflow;

    public WorkflowBuilder() {
        this.workflow = new Workflow();
    }

    public WorkflowBuilder id(Long id) {
        workflow.setId(id);
        return this;
    }

    public WorkflowBuilder name(String name) {
        workflow.setName(name);
        return this;
    }

    public WorkflowBuilder description(String description) {
        workflow.setDescription(description);
        return this;
    }

    public WorkflowBuilder version(String version) {
        workflow.setVersion(version);
        return this;
    }

    public WorkflowBuilder state(Integer state) {
        workflow.setState(state);
        return this;
    }

    public WorkflowBuilder nodes(List<IFlowNode> nodes) {
        workflow.setNodes(nodes);
        return this;
    }

    public WorkflowBuilder addNode(IFlowNode node) {
        workflow.getNodes().add(node);
        return this;
    }

    public WorkflowBuilder formMeta(FormMeta formMeta) {
        workflow.setFormMeta(formMeta);
        return this;
    }

    public Workflow build() {
        // 构建校验
        if (workflow.getName() == null || workflow.getName().isEmpty()) {
            throw new FlowValidationException("Workflow name is required");
        }
        return workflow;
    }
}
```

### 使用示例

```java
Workflow workflow = new WorkflowBuilder()
    .id(1L)
    .name("请假流程")
    .description("员工请假申请流程")
    .version("1.0")
    .state(1)
    .formMeta(formMeta)
    .nodes(nodes)
    .build();
```

## BaseNodeBuilder

```java
package com.codingapi.flow.builder;

/**
 * 节点构建器基类 - 模板方法模式
 */
public abstract class BaseNodeBuilder<T extends IFlowNode> {

    protected T node;

    protected BaseNodeBuilder() {
        this.node = createNode();
    }

    /**
     * 创建节点 - 子类实现
     */
    protected abstract T createNode();

    public T getNode() {
        return node;
    }

    public BaseNodeBuilder<T> nodeId(String nodeId) {
        node.setNodeId(nodeId);
        return this;
    }

    public BaseNodeBuilder<T> nodeName(String nodeName) {
        node.setNodeName(nodeName);
        return this;
    }

    public BaseNodeBuilder<T> order(int order) {
        node.setOrder(order);
        return this;
    }

    protected void validate() {
        if (node.getNodeId() == null || node.getNodeId().isEmpty()) {
            throw new FlowValidationException("Node ID is required");
        }
    }

    public T build() {
        validate();
        return node;
    }
}
```

## ActionBuilder

```java
package com.codingapi.flow.builder;

/**
 * 动作构建器
 */
public class ActionBuilder {

    private final List<IFlowAction> actions = new ArrayList<>();

    public ActionBuilder addAction(ActionType type, long actionId, String name) {
        IFlowAction action = FlowActionFactory.getAction(type);
        actions.add(action);
        return this;
    }

    public ActionBuilder addPassAction(long actionId, String name) {
        return addAction(ActionType.Pass, actionId, name);
    }

    public ActionBuilder addRejectAction(long actionId, String name) {
        return addAction(ActionType.Reject, actionId, name);
    }

    public ActionBuilder addReturnAction(long actionId, String name) {
        return addAction(ActionType.Return, actionId, name);
    }

    public List<IFlowAction> build() {
        return Collections.unmodifiableList(actions);
    }
}
```

## NodeStrategyBuilder

```java
package com.codingapi.flow.builder;

/**
 * 节点策略构建器
 */
public class NodeStrategyBuilder {

    private final List<INodeStrategy> strategies = new ArrayList<>();

    public NodeStrategyBuilder addStrategy(String type, StrategyConfig config) {
        INodeStrategy strategy = NodeStrategyFactory.getStrategy(type);
        strategy.setConfig(config);
        strategies.add(strategy);
        return this;
    }

    public NodeStrategyBuilder addTimeoutStrategy(long timeoutMillis) {
        StrategyConfig config = new StrategyConfig();
        config.put("timeoutMillis", timeoutMillis);
        return addStrategy(TimeoutStrategy.TYPE, config);
    }

    public NodeStrategyBuilder addMultiOperatorAuditStrategy(int approvalCount) {
        StrategyConfig config = new StrategyConfig();
        config.put("approvalCount", approvalCount);
        return addStrategy(MultiOperatorAuditStrategy.TYPE, config);
    }

    public List<INodeStrategy> build() {
        return Collections.unmodifiableList(strategies);
    }
}
```

## FormFieldPermissionsBuilder

```java
package com.codingapi.flow.builder;

/**
 * 表单字段权限构建器
 */
public class FormFieldPermissionsBuilder {

    private final Map<String, FieldPermission> permissions = new HashMap<>();

    public FormFieldPermissionsBuilder addReadableField(String fieldId) {
        permissions.put(fieldId, FieldPermission.readable());
        return this;
    }

    public FormFieldPermissionsBuilder addWritableField(String fieldId) {
        permissions.put(fieldId, FieldPermission.writable());
        return this;
    }

    public FormFieldPermissionsBuilder addHiddenField(String fieldId) {
        permissions.put(fieldId, FieldPermission.hidden());
        return this;
    }

    public Map<String, FieldPermission> build() {
        return Collections.unmodifiableMap(permissions);
    }
}
```

```
     │
     ├──► 构建节点策略

     └──► 构建节点动作
              │

              │
              ▼

              │
              ▼

```
