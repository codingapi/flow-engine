# 11 - 网关与上下文

## 模块概述

网关模块定义了与外部系统集成的接口，是外部用户系统接入工作流引擎的防腐层。

## FlowOperatorGateway

### 接口定义

```java
package com.codingapi.flow.gateway;

/**
 * 流程操作者网关接口
 * 用于接入外部用户系统，是外部用户系统与流程引擎之间的防腐层
 */
public interface FlowOperatorGateway {

    /**
     * 获取操作者
     */
    IFlowOperator get(long id);

    /**
     * 批量获取操作者
     */
    List<IFlowOperator> findByIds(List<Long> ids);
}
```

### 示例实现

```java
package com.codingapi.example.gateway;

@Component
public class FlowOperatorGatewayImpl implements FlowOperatorGateway {

    @Autowired
    private UserRepository userRepository;

    @Override
    public IFlowOperator get(long id) {
        User user = userRepository.findById(id);
        if (user == null) {
            return null;
        }
        return convertToOperator(user);
    }

    @Override
    public List<IFlowOperator> findByIds(List<Long> ids) {
        List<User> users = userRepository.findAllById(ids);
        return users.stream()
            .map(this::convertToOperator)
            .collect(Collectors.toList());
    }

    private IFlowOperator convertToOperator(User user) {
        return new IFlowOperator() {
            @Override
            public long getId() { return user.getId(); }

            @Override
            public String getName() { return user.getName(); }

            @Override
            public String getUsername() { return user.getUsername(); }

            @Override
            public Long getDeptId() { return user.getDeptId(); }

            @Override
            public Long getPostId() { return user.getPostId(); }

            @Override
            public boolean isFlowAdmin() {
                return "FLOW_ADMIN".equals(user.getRole());
            }

            @Override
            public boolean hasPermission(ActionType actionType) {
                return true;
            }
        };
    }
}
```

## GatewayContext

```java
package com.codingapi.flow.context;

/**
 * 网关上下文
 * 提供全局访问网关实例的单例访问点 - 门面模式
 */
public class GatewayContext {

    private static final GatewayContext INSTANCE = new GatewayContext();

    private FlowOperatorGateway operatorGateway;
    private IRepositoryHolder repositoryHolder;
    private IScriptExecutor scriptExecutor;

    private GatewayContext() {}

    public static GatewayContext getInstance() {
        return INSTANCE;
    }

    @PostConstruct
    public void init() {
        this.operatorGateway = ApplicationContextHolder.getBean(FlowOperatorGateway.class);
        this.repositoryHolder = ApplicationContextHolder.getBean(IRepositoryHolder.class);
        this.scriptExecutor = ApplicationContextHolder.getBean(IScriptExecutor.class);
    }

    public IFlowOperator getOperator(long id) {
        return operatorGateway.get(id);
    }

    public List<IFlowOperator> getOperators(List<Long> ids) {
        return operatorGateway.findByIds(ids);
    }

    public WorkflowRepository getWorkflowRepository() {
        return repositoryHolder.getWorkflowRepository();
    }

    public FlowRecordRepository getFlowRecordRepository() {
        return repositoryHolder.getFlowRecordRepository();
    }
}
```

### 事件类型

| 事件类 | 类型标识 | 说明 |
|--------|----------|------|
| `FlowRecordTodoEvent` | "record.todo" | 待办事件 |
| `FlowRecordDoneEvent` | "record.done" | 已办事件 |
| `FlowRecordCreateEvent` | "record.create" | 记录创建事件 |
| `FlowRecordTerminateEvent` | "record.terminate" | 记录终止事件 |
