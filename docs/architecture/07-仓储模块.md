# 07 - 仓储模块

## 模块概述

**包路径**: `com.codingapi.flow.repository`

仓储模块定义了数据访问接口，由 `flow-engine-starter-infra` 模块实现。

### 实现示例

```java
package com.codingapi.flow.infra.repository.impl;

@Repository
public class WorkflowRepositoryImpl implements WorkflowRepository {

    @Autowired
    private WorkflowEntityRepository entityRepository;

    @Autowired
    private WorkflowConvertor convertor;

    @Override
    public Workflow save(Workflow workflow) {
        WorkflowEntity entity = convertor.toEntity(workflow);
        entity = entityRepository.save(entity);
        return convertor.toWorkflow(entity);
    }

    @Override
    public Workflow findById(Long id) {
        return entityRepository.findById(id)
            .map(convertor::toWorkflow)
            .orElse(null);
    }
}
```

### 转换器模式

```java
package com.codingapi.flow.infra.convert;

@Component
public class WorkflowConvertor {

    /**
     * 实体转领域对象
     */
    public Workflow toWorkflow(WorkflowEntity entity) {
        if (entity == null) {
            return null;
        }
        Workflow workflow = new Workflow();
        workflow.setId(entity.getId());
        workflow.setName(entity.getName());
        workflow.setDescription(entity.getDescription());
        workflow.setState(entity.getState());
        workflow.setCreateTime(entity.getCreateTime());
        workflow.setUpdateTime(entity.getUpdateTime());
        return workflow;
    }

    /**
     * 领域对象转实体
     */
    public WorkflowEntity toEntity(Workflow workflow) {
        if (workflow == null) {
            return null;
        }
        WorkflowEntity entity = new WorkflowEntity();
        entity.setId(workflow.getId());
        entity.setName(workflow.getName());
        entity.setDescription(workflow.getDescription());
        entity.setState(workflow.getState());
        return entity;
    }
}
```
