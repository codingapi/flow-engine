# 12 - 数据库设计

## 概述

Flow Engine 支持 **H2** 内存数据库和**达梦**数据库，采用 Spring Data JPA 进行持久化。

## 数据库配置

### H2 数据库 (开发环境)

```yaml
spring:
  datasource:
    url: jdbc:h2:./example.db;DB_CLOSE_DELAY=-1;MODE=MySQL
    driver-class-name: org.h2.Driver
  jpa:
    database-platform: org.hibernate.dialect.H2Dialect
```

### 达梦数据库 (生产环境)

```yaml
spring:
  datasource:
    url: jdbc:dm://localhost:5236?schema=DAMENG
    driver-class-name: dm.jdbc.driver.DmDriver
  jpa:
    database-platform: com.codingapi.example.dialect.HydDmDialect
```

## 实体设计

### WorkflowEntity

```java
@Entity
@Table(name = "t_flow_workflow")
public class WorkflowEntity {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(nullable = false)
    private String name;

    private String description;

    @Column(nullable = false)
    private Integer state;  // 0-禁用, 1-启用

    private LocalDateTime createTime;
    private LocalDateTime updateTime;
}
```

### WorkflowVersionEntity

```java
@Entity
@Table(name = "t_flow_workflow_version")
public class WorkflowVersionEntity {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(nullable = false)
    private Long workflowId;

    @Column(nullable = false)
    private String version;

    @Column(columnDefinition = "TEXT")
    private String formMeta;  // JSON

    @Column(columnDefinition = "TEXT")
    private String nodeMetas;  // JSON

    @Column(columnDefinition = "TEXT")
    private String blocks;  // JSON

    @Column(nullable = false)
    private Integer state;  // 0-禁用, 1-启用

    private LocalDateTime createTime;
}
```

### FlowRecordEntity

```java
@Entity
@Table(name = "t_flow_record")
public class FlowRecordEntity {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(nullable = false)
    private Long workflowId;

    private Long versionId;

    private String businessId;

    @Column(columnDefinition = "TEXT")
    private String formData;  // JSON

    @Column(nullable = false)
    private Integer state;

    private String currentNodeId;

    private Long initOperatorId;

    private LocalDateTime createTime;
    private LocalDateTime updateTime;
}
```

## 数据库方言

### HydDmDialect

```java
package com.codingapi.example.dialect;

import org.hibernate.dialect.Dialect;
import org.hibernate.dialect.identity.IdentityColumnSupport;
import org.hibernate.dialect.identity.IdentityColumnSupportImpl;

public class HydDmDialect extends Dialect {

    @Override
    public IdentityColumnSupport getIdentityColumnSupport() {
        return new HydDmIdentityColumnSupport();
    }

    @Override
    public String getSequenceNextValString(String sequenceName) {
        return "select " + sequenceName + ".nextval from dual";
    }
}

public class HydDmIdentityColumnSupport extends IdentityColumnSupportImpl {

    @Override
    public boolean supportsIdentityColumns() {
        return true;
    }

    @Override
    public String getIdentitySelectString(String table, String column, int type) {
        return "select identity_val from dual";
    }

    @Override
    public String getIdentityColumnString(int type) {
        return "identity";
    }
}
```

```

       │
       │ 1:N
       ▼
t_flow_workflow_version
       │
       │ 1:N
       ▼
t_flow_record ─────────────────┐
       │                       │
       │ 1:N                   │ N:1
       ▼                       ▼
t_flow_todo_record      t_flow_operator_assignment
       │
       │ 1:1
       ▼
t_flow_form_data (JSON)

t_flow_record ──► t_delay_task ──► (触发器)
       │
       │ 1:N
       ▼
t_parallel_control
```
