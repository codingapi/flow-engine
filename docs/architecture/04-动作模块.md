# 04 - 动作模块

## 模块概述

**包路径**: `com.codingapi.flow.action`

动作是节点执行的具体操作，如通过、拒绝、保存等。每个节点可以绑定多个动作。

## 核心接口

### IFlowAction

```java
package com.codingapi.flow.action;

/**
 * 流程动作接口
 */
public interface IFlowAction {

    /**
     * 获取动作类型
     */
    ActionType getType();

    /**
     * 获取动作ID
     */
    long getActionId();

    /**
     * 获取动作名称
     */
    String getName();

    /**
     * 执行动作
     * @param session 流程会话
     * @param context 流程上下文
     */
    void execute(FlowSession session, FlowContext context);
}
```

### ActionDisplay

## BaseAction 抽象类

```java
package com.codingapi.flow.action;

/**
 * 动作基类 - 模板方法模式
 */
public abstract class BaseAction implements IFlowAction {

    @Override
    public final void execute(FlowSession session, FlowContext context) {
        // 1. 前置验证
        validate(session, context);

        // 2. 执行前处理
        beforeExecute(session, context);

        // 3. 执行核心逻辑
        doExecute(session, context);

        // 4. 执行后处理
        afterExecute(session, context);
    }

    /**
     * 核心执行逻辑 - 子类实现
     */
    protected abstract void doExecute(FlowSession session, FlowContext context);

    /**
     * 前置验证
     */
    protected void validate(FlowSession session, FlowContext context) {
        // 默认验证逻辑
    }

    /**
     * 执行前处理
     */
    protected void beforeExecute(FlowSession session, FlowContext context) {
    }

    /**
     * 执行后处理
     */
    protected void afterExecute(FlowSession session, FlowContext context) {
    }
}
```

## 动作实现类

| 类名 | 说明 | 核心逻辑 |
|------|------|----------|
| `PassAction` | 通过动作 | 流程流转到下一节点 |
| `RejectAction` | 拒绝动作 | 流程结束或回退 |
| `SaveAction` | 保存动作 | 仅保存表单数据 |
| `ReturnAction` | 退回动作 | 退回到指定节点 |
| `AddAuditAction` | 加签动作 | 添加审批人 |
| `DelegateAction` | 委托动作 | 将任务委托他人 |
| `TransferAction` | 转办动作 | 转给其他人处理 |
| `CustomAction` | 自定义动作 | 执行自定义脚本 |

### PassAction 示例

```java
package com.codingapi.flow.action;

public class PassAction extends BaseAction {

    @Override
    protected void doExecute(FlowSession session, FlowContext context) {
        // 1. 更新流程记录状态
        flowRecordService.markDone(session.getRecordId());

        // 2. 创建下一节点的待办记录
        IFlowNode nextNode = findNextNode(session);
        flowRecordService.createTodoRecord(nextNode, session);

        // 3. 记录操作日志
        recordService.saveActionLog(session, this);
    }
}
```

## 动作工厂

### FlowActionFactory

```java
package com.codingapi.flow.action.factory;

public class FlowActionFactory {

    private static final Map<ActionType, IFlowAction> ACTIONS = new ConcurrentHashMap<>();

    static {
        ACTIONS.put(ActionType.Pass, new PassAction());
        ACTIONS.put(ActionType.Reject, new RejectAction());
        ACTIONS.put(ActionType.Save, new SaveAction());
        ACTIONS.put(ActionType.Return, new ReturnAction());
        ACTIONS.put(ActionType.AddAudit, new AddAuditAction());
        ACTIONS.put(ActionType.Delegate, new DelegateAction());
        ACTIONS.put(ActionType.Transfer, new TransferAction());
        ACTIONS.put(ActionType.Custom, new CustomAction());
    }

    public static IFlowAction getAction(ActionType type) {
        IFlowAction action = ACTIONS.get(type);
        if (action == null) {
            throw new FlowException("Unsupported action type: " + type);
        }
        return action;
    }
}
```

## 动作执行流程

```
Client
   │
   ▼
FlowRecordController POST /action
   │
   ▼
ActionManager.executeAction(recordId, actionType)
   │
   ▼
FlowActionFactory.getAction(actionType)
   │
   ▼
BaseAction.execute(session, context)
   │
   ├──► validate()
   ├──► beforeExecute()
   ├──► doExecute() ──► FlowService
   └──► afterExecute()
```
