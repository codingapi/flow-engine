# 01 - 项目概览

## 项目背景

Flow Engine 是一个**企业级工作流引擎**，基于 Java 17 和 Spring Boot 3.5.9 构建。提供可视化流程设计、动态表单配置、多节点类型流转以及脚本扩展功能。

### 核心特性

- **可视化流程设计**：支持拖拽式流程设计
- **动态表单配置**：灵活定义表单字段和权限
- **多节点类型流转**：19 种节点类型支持复杂业务流程
- **脚本扩展**：Groovy 脚本支持自定义业务逻辑

## 项目结构

```
flow-engine/
├── flow-engine-framework/      # 核心引擎框架
│   └── com.codingapi.flow/
│       ├── node/             # 节点模块
│       ├── action/           # 动作模块
│       ├── strategy/         # 策略模块
│       ├── script/           # 脚本模块
│       ├── repository/       # 仓储模块
│       ├── builder/          # 构建器模块
│       ├── manager/          # 管理器模块
│       └── service/          # 服务模块
├── flow-engine-starter/       # Spring Boot 自动配置
├── flow-engine-starter-api/   # REST API 层
├── flow-engine-starter-infra/ # 持久化层
├── flow-engine-starter-query/ # 查询层
├── flow-engine-example/      # 示例应用
└── flow-frontend/             # 前端子模块 (git submodule)
```
