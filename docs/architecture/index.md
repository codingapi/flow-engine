## 模块依赖关系

```
flow-engine-example
       │
       ├──► flow-engine-starter-api
       ├──► flow-engine-starter-infra
       └──► flow-engine-starter-query
                  │
                  ▼
         flow-engine-starter
                  │
                  ▼
         flow-engine-framework
                  │
                  ▼
       flow-engine-starter-infra
```

## flow-engine-framework 八层架构

```
┌─────────────────────────────────────────────────────────────┐
│  工作流层  │  Workflow, WorkflowVersion                      │
├─────────────────────────────────────────────────────────────┤
│  节点层    │  StartNode, ApprovalNode, RouterNode 等 19 种   │
├─────────────────────────────────────────────────────────────┤
│  动作层    │  PassAction, RejectAction 等 8 种               │
├─────────────────────────────────────────────────────────────┤
│  记录层    │  FlowRecord, FlowTodoRecord                     │
├─────────────────────────────────────────────────────────────┤
│  会话层    │  FlowSession                                   │
├─────────────────────────────────────────────────────────────┤
│  管理器层  │  ActionManager, OperatorManager, NodeStrategyManager │
├─────────────────────────────────────────────────────────────┤
│  策略层    │  NodeStrategy, WorkflowStrategy                 │
├─────────────────────────────────────────────────────────────┤
│  脚本层    │  ScriptRuntimeContext, GroovyScript             │
└─────────────────────────────────────────────────────────────┘
```
